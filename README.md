# ssh-terminal-chat

a terminal-based chat application accessible via ssh, built with spring boot.

## features (planned)

- ssh server for remote access
- user auth and account creation
- real-time chat functionality
- cli for user interaction

## technologies

- java 17
- spring boot 3.3.3
- apache mina sshd
- spring security
- spring data jpa
- h2 database (for dev)
